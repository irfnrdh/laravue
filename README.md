# LaraVue

> Starter project dengan Laravel Mix dan Vue Js

### Membuat Project

`laravel new laravue`
`npm install`
`npm install vue-router`

### Menjalankan Project

`php artisan serve`
`npm run watch`

### Membuat login & Register

`php artisan make:auth`

### Migrasi

#### Migrasi Data ke Database

`php artisan migrate`

#### Migrasi Ulang Setelah Error

`php artisan migrate:fresh`
